#README – Projet Game Arena
##Présentation générale

Game Arena est un site web dédié à la présentation de jeux vidéo, à la mise en avant de personnages populaires, et à l’affichage d’achats in-game. 
Le site propose plusieurs sections accessibles via une navigation fixe, une expérience immersive avec effets visuels, et un mode clair/sombre. Le projet répond aux différents critères pédagogiques imposés dans le sujet d’examen.

##Technologies utilisées

Le projet est réalisé en utilisant :

HTML5 pour la structure sémantique du site.

CSS3 pour la mise en forme, l’animation, et la gestion des thèmes (dark/light).

JavaScript pour les interactions et le système d’envoi d’e-mail via EmailJS.

Aucune bibliothèque externe n’est utilisée pour l’interface, hormis Google Fonts et EmailJS.

##Fonctionnalités principales

###Navigation fixe avec icônes SVG animés
La barre de navigation reste accessible en permanence et affiche des descriptions au survol.

###Mode clair / mode sombre
Un système de changement de thème modifie dynamiquement les variables CSS via JavaScript.

###Section Home animée
Texte animé, boutons interactifs, personnage flottant.

###Section Jeux (Game)
Présentation de plusieurs jeux populaires, chaque bloc contenant une description et un logo.

###Section Achats (Shop)
Tables d’achats in-game : Free Fire, PUBG, Fortnite, Valorant, Marvel Rivals et COD: Black OPS 7.
Boutons d’achat redirigeant vers l’adresse e-mail Game Arena.

###Section Personnages (Topperso)
Classement de trois personnages avec image, logo, description, et vidéo intégrée depuis YouTube.

###Section Contact
Réseaux sociaux cliquables, image illustrative, et formulaire de contact fonctionnel avec EmailJS.

##Responsive Design
Le site s’adapte aux différents écrans.

##Back-end (EmailJS)
Le projet utilise EmailJS pour gérer l’envoi d’e-mails via le formulaire de contact.
Le script est intégré dans script.js et se charge de traiter les formulaires sans serveur supplémentaire.
